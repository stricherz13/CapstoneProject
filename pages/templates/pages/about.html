About STL Tax Auction Map

STL Tax Auction Map is a tool to help real estate investors in the St Louis area identify potential investment properties from land tax auctions. STL Tax Auction Map maps the available properties and uses publicly available information to allow the user to query the properties based on location and property features. .


Questions & Comments

Attributions

All data used in STL Tax Auction Map are publicly available datasets from city, county, and state government offices.


Technology Stack

While I hope STL Tax Auction Map is useful, I realize it is a very niche tool. I use STL Tax Auction Map as an opportunity to work with technologies I want to learn or have a preference for that I don't necessarily get to use at my day job. Here is what the site is currently built with:

The application is written in Django.

Geoprocessing of the data is handled in QGIS. On the backend the site uses PostgreSQL database with the PostGIS extension to handle the vector data. The vector data is served as a Web Feature Service using Geoserver.

The frontend is built with Django template language. Styling is handled using bootstrap. The JavaScript map library is provided by Leaflet
